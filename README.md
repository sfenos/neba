# Neba

**Neba** is a high-performance, dynamically-typed interpreted language with optional static typing.

## Status
- [x] v0.1.0 — Lexer
- [ ] v0.1.1 — Parser + AST
- [ ] v0.1.2 — Tree-walking interpreter
- [ ] v0.1.3 — REPL

## Quick Start
```bash
cargo run --bin neba -- tests/lexer_tests/basic_tokens.neba
```
